# Mediverse 🏥

Mediverse is a comprehensive healthcare management system designed to streamline hospital operations and improve patient care through a structured and database-driven platform.

🏆 **Achievement:** Ranked **4th place among 120+ teams** in the university project showcase.

The system integrates multiple healthcare services such as patient management, doctor scheduling, pharmacy services, laboratory reports, and ambulance services into a single unified platform.

---

## 📖 Overview

Mediverse was developed as a full-scale academic project to demonstrate the practical implementation of database systems, software engineering concepts, and system design. The platform focuses on organizing healthcare data and simplifying hospital workflows through a multi-role system.

Each user role operates through its own **secure dashboard and control panel**, allowing efficient management of healthcare services.

---

## 🚀 Key Features

- Patient registration and profile management
- Doctor appointment and scheduling system
- Digital prescription and medical record management
- Laboratory test management and report upload
- Pharmacy medicine management
- Ambulance service request and management
- Role-based dashboards and access control
- Structured healthcare workflow management

---

## 👥 User Roles & Dashboards

Mediverse supports multiple user roles. Each role has its own dedicated dashboard to manage specific operations.

### 🧑‍⚕️ Patient
- Register and manage personal profile
- Book doctor appointments
- View prescriptions and medical records
- Access lab reports

### 👨‍⚕️ Doctor
- Manage patient appointments
- Access patient medical history
- Provide prescriptions and treatment notes

### 💊 Pharmacy
- Manage medicine inventory
- Process doctor prescriptions
- Track medicine availability

### 🧪 Lab Technician
- Receive lab test requests
- Upload and manage test reports
- Maintain laboratory records

### 🚑 Ambulance Service
- Manage ambulance availability
- Handle emergency transport requests

### ⚙️ System Admin
- Manage overall platform settings
- Monitor system operations

### 🏥 Hospital Super Admin
- Control hospital-level operations
- Manage hospital resources and services

### 🏢 Hospital Sub Admin
- Assist in hospital administration
- Manage staff and service records
